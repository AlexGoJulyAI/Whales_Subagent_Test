Direction C · Master-Detail Column · User Journey · GoJuly AI — Previous Conversations
The Problem

To find any past conversation, users must click into every attempt and read the full transcript. There is no way to compare attempts across challenges — each is siloed behind an accordion. A researcher wanting to reference a specific exchange must navigate away, then come back. There is no quick-reference view.

Solution

A fixed left column (280px) lists all challenges with scenario names, attempt counts, and aggregate outcome summaries. Selecting a challenge updates the right-column feed without any page navigation. Attempt entries can expand inline to show the first exchange — a quick-reference that never leaves the page. A search bar in the left column filters both the challenge list and the attempt feed simultaneously.
