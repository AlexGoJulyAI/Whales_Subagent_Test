Direction C · See It First — User Journey
The Problem

Parents completing safety setup are configuring abstract policies for a future experience they can't picture. When the effort feels disconnected from an outcome they care about, motivation drops — especially for settings that feel unfamiliar or overly technical.

This Direction

The parent sees their child's name on their own dashboard before doing any setup. A focused sheet overlay guides configuration while the destination is always visible in the background. Every setting they choose feels like building something real — for a specific child they love, not for an abstract system.
