TreeHouse / SafeChat — User Journey
Direction C — Trust Layer
Problem
Parents who trust their instincts are forced through a 5-step setup anyway. Parents who want details have no way to access them without doing the whole flow.
Solution
A single overview panel shows all 5 categories defaulted to Balanced — ready immediately. Experts drill into any category via bottom sheet. Everyone else taps "Done" in under 30 seconds.
