TreeHouse / SafeChat — User Journey
Direction A — Confident Setup
Problem
Parents face 21 AI settings with no way to know which matter. The interface treats every option as equally important, creating decision paralysis.
Solution
Settings are grouped into 5 decision cards. Subcategory details collapse by default — power users tap to expand, others stay on the headline tier choice. Every setting gives a one-line consequence, not a description.
