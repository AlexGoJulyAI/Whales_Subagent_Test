Direction A — Three-Pane Drill-Down
Problem
The conversations list only shows dates — there's no way to tell what a conversation covered without clicking into each one. Finding a specific attempt means expanding three separate sections, then opening each wrong row to check. A user with 8+ attempts per challenge has no way to scan efficiently.
Solution — Direction A
A three-pane layout keeps challenges, attempts, and a conversation preview simultaneously visible at all times. Every attempt row leads with its AI-generated topic name — clicking a row loads the preview inline so the user can confirm they found the right conversation before opening it fully.
