GoJuly · User Journey
Direction B — Command Search
Problem

The conversations list only shows dates — there's no way to tell what a conversation covered without clicking into each one. With no search bar and three separate accordion sections to expand, finding a specific conversation requires opening every wrong one first.

Solution — Command Search

A dominant search bar sits at the top of the page as the primary UI element — the first thing a user's eye lands on. Below it, every attempt from every challenge loads immediately with its AI-generated topic name. Typing a keyword filters the list in real-time. No accordion clicks, no date-memory required.
