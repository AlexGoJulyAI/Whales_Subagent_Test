Direction B · Configuration Hub — User Journey
The Problem

A linear wizard with 21 steps gives parents no sense of how much remains. When the next screen looks identical to the last, drop-off happens — not because parents don't care, but because progress is invisible. Parents who stop at step 14 have no idea they're 2/3 done.

This Direction

A hub screen shows all five safety categories as cards. Parents see the whole map before they start, choose any category in any order, and watch completed cards turn green. The same hub is the settings editor on the dashboard — zero re-learning when a parent returns months later to change a setting.
