Component Specs & Interaction Notes
1 Challenge Summary Tiles
Purpose Replace accordion cards. Show attempt count (large display number), success rate, and last attempt date before interacting with any individual row.
Layout 3-column grid, equal width, 10px gap. Each tile: white bg, 1.5px border #dadee7, 12px border-radius, 14/16px padding.
Count 46px / weight 800 / letter-spacing −0.04em. Visual anchor for the tile — highest weight element on the page after the page title.
Active state bg #083386, all text white/muted-white. box-shadow 0 2px 18px rgba(8,51,134,.30). Rule line: rgba(255,255,255,.18). Guarded % shows as rgb(162,232,165) so it pops on navy.
Hover (inactive) box-shadow 0 2px 10px rgba(8,51,134,.09), border-color #a0b4d8. 150ms ease transition.
Toggle logic Clicking an inactive tile activates it (filters list) and deactivates any previously active tile. Clicking the already-active tile deactivates it and restores the full unfiltered list.
Tile data C1: 8 attempts, 50% guarded, last Mar 9 2026  ·  C2: 4 attempts, 75% guarded, last Mar 9 2026  ·  C3: 2 attempts, 50% guarded, last Mar 9 2026
Guarded % C1 navy #083386, C2 teal #0D7490, C3 purple #6D28D9 — color-coded to match challenge pill colors when inactive.
2 AI-Generated Topic Name (Primary Row Label)
Typography 13.5px / weight 600 / color #1E2A40. Truncates with ellipsis at cell edge. Highest-weight text on each row.
Replaces The current date-only label (which provides no conversational context). Date moves to its own dedicated column in muted #888888 11.5px.
Generation AI-inferred from the first 3–5 conversation turns. Stored server-side on first read. Falls back to "Conversation [date]" if generation fails or is pending.
3 Search Bar
Position Full-width between tiles row and attempt list. 44px tall, 8px border-radius.
Placeholder "Filter by topic or keyword..."
Focus state Border 1.5px #083386 (shown as default in this mockup since it is the primary affordance). Icon color also #083386.
Scope & logic Filters the unified list by topic name substring match. When a tile filter is active, search applies within that filtered set (AND logic). Clearing search text removes the keyword filter without affecting the tile filter.
Icon 16×16 SVG magnifier, left-padded 14px. Color #083386 when focused / #aaaaaa at rest.
4 Unified Attempt List
Default view All 14 attempts (8+4+2), sorted by date descending. The C1/C2/C3 badge on every row identifies the challenge — no section headers needed.
Filtered view When a tile is active, list shows only that challenge's rows. Filter pill appears beside search bar. Footer: "Showing N of M attempts for Challenge X".
Row 50px min-height, 14px horizontal padding, border-bottom #eaecf0. Last row no border. Hover: bg #f4f6fa + 3px navy left accent (::before). Click: opens conversation in new tab.
Column widths Topic: flex 1  ·  Ch: 40px  ·  Date: 90px  ·  Attack Status: 90px  ·  Progress: 94px  ·  Share: 44px
Container 1.5px border #dadee7, 12px border-radius, white bg, overflow hidden. Column header row: 34px / bg #f4f6fa / 9.5px uppercase headers.
Share btn 26×26px, border #dadee7, hover: border+icon to #083386 + bg #ebf0f7. Copies shareable link to clipboard — independent of row click.

5 Active Tile Filter & Filter Pill
Activation Click any challenge tile. That tile switches to navy bg + white text (active state). Only one tile can be active at a time.
Filter pill Appears to the right of the search bar when a filter is active. Label: "Showing: Challenge N". bg #dbeafe, border #93c5fd, text #083386, 12px font-weight 700. Fades in with 100ms opacity transition.
Clear Three equivalent affordances: (a) × icon in filter pill, (b) "Clear filter →" footer link, (c) click the active tile again. All three restore full unfiltered list and remove pill.
Animation Tile bg/text transitions 150ms ease. List rows fade in 130ms ease on filter change. Filter pill: 100ms opacity fade.
