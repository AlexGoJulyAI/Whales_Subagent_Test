Direction at a Glance
  • Narrator name is the primary label on every list row — above title, in brand-blue (#1479cf)
  • Single sticky chip row handles all filtering without secondary UI layer
  • Most token-faithful to Authors page — lowest implementation risk, highest stakeholder recognizability
Strategy

Direction A exists because the §3 primary problem is not that filters are hard to use — it is that narrator identity is invisible in the current page layout. Placing the narrator name first on every row directly solves the discovery deficit identified in §3 and §12 Task 1 (find audiobooks by a specific narrator). The single chip row honors §4’s secondary metric (filter controls visible) while preserving the Authors page’s spatial contract, minimizing the stakeholder delta required for approval.

  2. Results counter — confirms the catalog is substantive (847 results)
