Audible Narrators Feature

Platform: Audible.com Surface: Browse Nav + /narrators page Date: 2026-04-25 Status: Brief Complete — Design Phase Next

Problem & Solution

Problem

Audible has no way to browse by narrator in the main navigation. Listeners choose audiobooks heavily by narrator voice and style, but the Browse dropdown only surfaces genre-based and editorial lists. There is no narrator equivalent to the existing Authors page.

Solution

Add "Narrators" to the Browse nav's Popular Lists column, linking to a new /narrators page that mirrors the /authors architecture — with narrator-specific filter chips (genre specialty, voice style, language, most-listened).

Scope — Two Changes

Change 1

Browse Nav — Add "Narrators" to Popular Lists

Add "Narrators" as a 6th item to the Popular Lists column. No existing items are removed.

  • Bestsellers
  • Coming Soon
  • New Releases
  • Best of the Year
  • Best of #BookTok
  • Narrators NEW
Change 2

/narrators Page

New page at /narrators mirroring /authors. H1: "Narrators". Filter chips row with dimensions: genre specialty, voice style, language, most-listened. Content: book-list view or narrator directory (Q3 — see Design Directions below).

Design Directions (Q3 Open — All Three Explored in Mockups)

Direction A
Book-list View
Mirror of /authors. Each row = audiobook with 232×232px cover, title link, "Narrated by: [Name]", price. Filter chips narrow by narrator tags. Nearly identical pattern to /authors.
Delta: Minimal
Direction B
Narrator Directory
Grid of narrator profile cards: headshot/avatar, name, genre tags, "X audiobooks" count. Click → narrator catalog page. New pattern on Audible.
Delta: New pattern
Direction C
Narrator-First List
Hybrid: list view with narrator as primary entity. Name (H3) + inline top-3 book covers + genre/voice chips + "See all by [Name]" CTA.
Delta: Medium
